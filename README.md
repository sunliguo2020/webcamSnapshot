# webcamSnapshot - 网络摄像头批量截图工具

## 项目背景

维护着很多摄像头，资料一直很不准确。某天一个摄像头故障，通道名称描述不是很准确，无法很快找到这个摄像头。我想，在摄像头正常的时候，每个摄像头都能截一张图。那么维护的时候不是就很容易找到它了吗？手工截图很慢，就可以用 python 来解决这个问题。

一开始用摄像头厂商的 SDK 来截图，发现比较复杂。后来从网上找到了可以用 RTSP 协议，还可以用 ONVIF 协议来截图。

## 功能简介

### 支持的摄像头类型
- **海康威视** - 通过 RTSP 协议截图
- **大华** - 通过 RTSP 协议截图
- **ONVIF 协议** - 支持标准 ONVIF 协议的摄像头
- **电脑 USB 摄像头** - 本地摄像头批量截图

### 图形界面功能
- **CSV 文件管理** - 选择并预览摄像头 IP 和密码列表（CSV 格式）
- **截图保存目录** - 自定义截图保存位置，自动按日期和类型分类
- **时间水印** - 可选在截图上添加时间水印
- **暂停/继续** - 截图过程中可随时暂停和恢复
- **实时日志** - 显示运行日志，支持清空
- **打开截图目录** - 截图完成后一键打开保存目录

### CSV 文件格式
CSV 文件第一行为表头，至少包含 `ip` 和 `password` 字段，例如：

```csv
ip,password
192.168.1.100,admin123
192.168.1.101,password456
```
